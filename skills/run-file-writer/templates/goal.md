# Goal

## Task

Write the requested task in one or two clear paragraphs.

## Input Material

List the files, folders, messages, or other materials the controlled Codex must use.

## Output Requirement

List the files, folders, messages, or other outputs the controlled Codex must produce.

## Progress Requirement

Name the evidence that proves the controlled Codex is making progress. This can be a file, screen output, or another observable result.

## Completion Standard

State what proves the task is complete. Include the check command when a command can prove completion.
