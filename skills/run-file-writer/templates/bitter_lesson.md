# Bitter Lesson

Use this file for a reusable mistake or failure that should prevent repeated errors.

## Problem

State what went wrong.

## Evidence

List the file, command, log, or transcript evidence.

## Cause

State the direct cause. Do not blame uncertainty; name the missing check, wrong assumption, or bad rule.

## Prevention Rule

Write the rule that should prevent the same problem next time.

## Where To Apply

List the template, skill, spec, script, or workflow file that should include this rule.
